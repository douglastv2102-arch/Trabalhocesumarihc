# 🧠 Trabalho de Interação Humano-Computador (IHC)

## 📌 Tema

Protótipo de Tela de Cadastro de Produtos com Base nas Heurísticas de Nielsen

---

## 👨‍🎓 Disciplina

Interação Humano-Computador (IHC)

## 🏫 Instituição

UniCesumar

---

## 🎯 Objetivo do Trabalho

Este trabalho tem como objetivo desenvolver um protótipo de interface para cadastro de produtos, aplicando conceitos de usabilidade e experiência do usuário (UX), com base nas Heurísticas de Nielsen.

A proposta busca simular uma tela real de sistema administrativo (e-commerce), priorizando clareza, organização e feedback ao usuário.

---

## 🖥️ Descrição do Protótipo

Foi desenvolvido um protótipo de alta fidelidade (HI-FI) representando uma tela de cadastro de produtos em um painel administrativo de e-commerce.

A interface contém:

- 📦 Informações do produto (nome, SKU, código de barras, classe fiscal)
- 💰 Preços (valor de venda e promocional)
- 📝 Conteúdo (resumo e descrição completa)
- 🖼️ Imagens do produto (upload e galeria)
- 📏 Dimensões e peso
- 📊 Estoque (quantidade e alerta mínimo)
- 🏷️ Categorias (seleção por checkboxes)
- ⚙️ Status do produto (ativo e controle de estoque)

---

## 🎨 Características da Interface

- Tema escuro (dark mode moderno)
- Layout baseado em cards (organização modular)
- Hierarquia visual clara
- Inputs padronizados
- Uso de ícones e cores para reforço visual
- Interface inspirada em sistemas SaaS modernos

---

## 🧠 Heurísticas de Nielsen Aplicadas

### ✔ 1. Visibilidade do estado do sistema

O sistema mantém o usuário informado através de:

- Mensagens de sucesso (ex: “Produto salvo com sucesso”)
- Alertas visuais (ex: estoque baixo)
- Feedback em tempo real nos campos (validação com cores e ícones)

Isso permite ao usuário compreender claramente o resultado de suas ações.

---

### ✔ 2. Prevenção de erros

A interface foi projetada para evitar erros antes que eles ocorram, através de:

- Validação visual nos campos (cores e mensagens)
- Separação lógica das informações
- Uso de campos específicos (ex: preço, quantidade, SKU)

Essas estratégias reduzem a probabilidade de inserção de dados incorretos.

---

### ✔ 3. Reconhecimento em vez de memorização

O sistema facilita o uso ao apresentar todas as informações de forma visível:

- Labels claros e objetivos
- Categorias listadas
- Navegação lateral acessível
- Organização por seções

Dessa forma, o usuário não precisa memorizar informações para utilizar o sistema.
